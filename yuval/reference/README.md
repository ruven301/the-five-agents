# תיקיית ה-reference של יובל

שים/שימי כאן תמונות השראה לסגנון (PNG/JPG) — תמונות שמייצגות את השפה הוויזואלית
שאנחנו רוצים לשמור עליה: פלטת צבעים, קומפוזיציה, טון ואלמנטים חוזרים.

יובל סורק את התיקייה הזו בתחילת כל בקשת תמונה ומחלץ ממנה את הסגנון, כדי שכל
תמונה חדשה תהיה עקבית עם השאר.

קובץ אחד לכל דוגמה. אם התיקייה ריקה — יובל עובד עם ברירת מחדל ויזואלית נקייה.
