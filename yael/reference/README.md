# תיקיית הדוגמאות של יעל

שים/שימי כאן קובצי `.md` עם דוגמאות לטקסטים שכבר כתובים בסגנון הבית שלנו —
מאמרים, פוסטים או קטעים שאתם אוהבים.

יעל קוראת את כל הקבצים בתיקייה הזו בתחילת כל סשן עבודה, ולומדת מהם את הטון,
הקצב והניסוח שלנו לפני שהיא משכתבת מאמר חדש.

קובץ אחד לכל דוגמה. שם הקובץ חופשי (למשל `example-blog-post.md`).
