# yuval/reference/ — תיקיית תמונות ההשראה

> [!info] קובץ בפרויקט
> **נתיב:** `yuval/reference/` (מתועד דרך ה-`README.md` שבתוכה)
> **משויך ל:** יובל
> **סוג:** תיקיית reference ויזואלי

## מה התיקייה עושה
מכילה תמונות השראה (PNG/JPG) שמייצגות את השפה הוויזואלית של הפרויקט: פלטת צבעים,
קומפוזיציה, טון ואלמנטים חוזרים. יובל סורק אותה בתחילת כל בקשת תמונה ומחלץ ממנה את
הסגנון, כדי שכל תמונה חדשה תהיה עקבית עם השאר.

## פרטים עיקריים
- קובץ אחד לכל דוגמה.
- כרגע מכילה רק `README.md` — אם ריקה, יובל עובד עם ברירת מחדל ויזואלית נקייה.
