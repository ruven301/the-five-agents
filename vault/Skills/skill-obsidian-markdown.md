# obsidian-markdown — Obsidian Flavored Markdown

> [!info] קובץ בפרויקט
> **נתיב:** `.claude/skills/obsidian-markdown/SKILL.md`

## מה הקובץ עושה
מדריך ליצירה ועריכה של Obsidian Flavored Markdown תקין — ההרחבות של אובסידיאן מעבר
ל-CommonMark/GFM: wikilinks, embeds, callouts, properties (frontmatter), comments,
tags, math, mermaid ועוד. זהו ה-reference התחבירי לכל קבצי ה-vault.

## פרטים עיקריים
- **wikilinks:** `[[Note]]`, `[[Note|תצוגה]]`, `[[Note#כותרת]]`, `[[Note#^block]]`.
- **embeds:** `![[Note]]`, `![[image.png|300]]`, `![[doc.pdf#page=3]]`.
- **callouts:** `> [!note]`, `> [!warning] כותרת`, גרסה מתקפלת עם `-`.
- **properties:** frontmatter עם tags/aliases/cssclasses.
- הערה: ה-SKILL במקור הפנה לקובצי `references/` (PROPERTIES/EMBEDS/CALLOUTS) שלא היו
  קיימים בריפו — בהתקנה הוסרו ההפניות החסרות.
